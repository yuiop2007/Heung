display 속성

        -요소들을 어떻게 화면에 보여줄지 결정
        -속성의 처리값 : visibility(보여주기) / hudden(감추기)
        -none : 보여주지않음(제거)
        -inline : 글자(단어)만의 공간을 확보
        -block : 지정된 블록영역을 확보
        -inline-block : inline과 block의 합성
